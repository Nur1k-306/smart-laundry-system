# smart-laundry-system

Система управления умной прачечной представляет собой микросервисное приложение, предназначенное для автоматизации процессов бронирования стиральных машин, обработки оплаты и управления процессом стирки. Пользователь может просматривать доступные устройства, создавать бронирования, выполнять оплату и получать уведомления о статусе стирки.

Архитектура системы построена на основе микросервисного подхода, где каждый сервис отвечает за отдельную бизнес-область: управление пользователями, устройствами, бронированиями, платежами, сессиями стирки и уведомлениями. Взаимодействие между сервисами реализовано как синхронно через REST API, так и асинхронно с использованием брокера сообщений Kafka.

В качестве технологического стека используется Java 20 и Spring Boot. Для хранения данных применяются PostgreSQL (реляционные данные) и MongoDB (документные данные). Redis используется для кэширования, а MinIO — для хранения файлов и аналитических выгрузок. Вся инфраструктура разворачивается с помощью Docker Compose.

В системе реализован API Gateway, который выступает единой точкой входа, выполняет маршрутизацию запросов и проверку аутентификации на основе JWT. Также подключены инструменты наблюдаемости: Prometheus, Grafana и Jaeger.

В репозитории представлены архитектурные диаграммы (C4 и Sequence), демонстрирующие структуру системы и взаимодействие микросервисов.
